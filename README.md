## Tecnologias Utilizadas

* **Backend:** Python com o micro-framework Flask.
* **Frontend:** HTML5, CSS3 avançado (Flexbox/Grid, Glassmorphism, animações) e Vanilla JavaScript.
* **Design:** Layout One-Page dinâmico, interações assíncronas (Reveal Cards) e menu mobile em overlay.

## Principais Funcionalidades

* **Navegação Suave (Smooth Scroll):** Menu dinâmico que se adapta ao scroll da página (Top Bar branca para Glassmorphism transparente).
* **Cartões Interativos:** Seção de serviços com expansão sob demanda (Reveal Cards) para uma leitura mais limpa.
* **Integração de Contato:** Botão flutuante para atendimento via WhatsApp e formulário de contato integrado para captação de leads.
* **Responsividade Total:** Adaptado perfeitamente para dispositivos móveis, tablets e desktops.
